# 💳 Credit Card Financial Dashboard

## 📌 Project Overview

This project is an end-to-end Credit Card Financial Dashboard built using **MySQL** and **Power BI**. It transforms raw customer and transaction data into an interactive dashboard that provides valuable business insights through data analysis and visualization.

The dashboard helps monitor weekly and yearly performance, customer behavior, revenue trends, and key business metrics to support better decision-making.

---

## 🎯 Project Objective

The objective of this project is to develop a comprehensive Credit Card Financial Dashboard that provides real-time insights into key performance indicators (KPIs) and business trends. The dashboard enables stakeholders to effectively monitor credit card operations and analyze overall business performance.


## 🛠️ Tools & Technologies

* Power BI
* MySQL
* Microsoft Excel (CSV Files)
* DAX (Data Analysis Expressions)
* Git & GitHub

## 📂 Dataset

The project uses two datasets:

* Customer Details
* Credit Card Transaction Details

## ⚙️ Project Workflow

### Step 1: Data Import

* Imported CSV files into MySQL.
* Created database and tables.
* Loaded customer and transaction data into SQL.

### Step 2: Data Processing

* Cleaned and transformed the data.
* Created calculated columns using DAX.
* Built measures for revenue, weekly performance, and KPIs.

### Step 3: Dashboard Development

Designed an interactive Power BI dashboard to analyze:

* Revenue
* Interest Earned
* Transaction Amount
* Transaction Count
* Customer Demographics
* Card Categories
* Weekly Performance
* Year-to-Date (YTD) Performance

---

## 📊 Dashboard Features

The dashboard includes:

* Total Revenue
* Total Interest Earned
* Total Transaction Amount
* Total Transaction Count
* Revenue by Card Category
* Revenue by Customer Job
* Revenue by Age Group
* Revenue by Income Group
* Revenue by Gender
* Revenue by State
* Quarterly Revenue Trends
* Weekly Revenue Analysis
* Current Week vs Previous Week Comparison

---

## 📈 Key Business Insights

* Overall revenue reached **57M**.
* Total transaction amount reached **46M**.
* Total interest earned reached **8M**.
* Male customers generated higher revenue than female customers.
* Blue and Silver credit cards contributed to approximately **93%** of all transactions.
* Texas, New York, and California contributed around **68%** of total revenue.
* Overall card activation rate was **57.5%**.
* Overall delinquency rate was **6.06%**.
* Weekly dashboard enables easy comparison between current and previous week performance.
